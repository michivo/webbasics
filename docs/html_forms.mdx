**Forms** bzw. **Formulare** sind ein wichtiges Element, um Benutzer\*innen die Interaktion mit Ihrer Seite zu ermöglichen, so z.B. bei
- Suchmasken, in denen Suchbegriffe eingegeben werden
- Anmeldeformularen, bei denen Benutzername und Passwort angegeben wird
- Kontakformularen
- ...

All diesen Formularen ist gemein, dass Benutzer\*innen Daten eingeben, welche dann an einen Server geschickt werden. Der Client erwartet sich eine Antwort und zeigt diese der\*m Benutzer\*in an.

Ein Formular wird immer von einem `form`-Tag umschlossen, das in der Regel folgende Attribute hat:
- `action` um anzugeben, an welche URL die Formularinhalte geschickt werden sollen
- `method` um die HTTP-Methode festzulegen, mit der der Request an die im `action`-Attribut angegebene URL geschickt werden soll (also `post` oder `get`)
- `target` um anzugeben, in welchem Kontext die Antwort des Servers angezeigt werden soll (`_self`, `_blank`, `_parent`, `_top` oder der jeweilige Framename)
