übersichtlich dargestellt und dokumentiert. In diesem Skriptum gibt es Beispiele für die bei Mozilla angeführten
Elemente. 

Die Beispielseiten sind optisch noch nicht wirklich ansprechend. Klarerweise würden Sie, wenn Sie
wirklich eine Webseite bauen, Styleinformationen für die entsprechenden Elemente hinterlegen.
