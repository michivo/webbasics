Elemente für eingebettete Inhalte#

In eine Seite können auch externe Inhalte eingebettet werden. Externe Inhalte können einerseits von externe Webseiten kommen (z.B. Straßenkarten von Google Maps oder OpenStreetMaps, Videos von YouTube oder anderen Anbietern, ...), andererseits können auch Inhalte eingebettet werden, die durch Browser-Plugins dargestellt werden können. Früher waren solche Plugins weit verbreitet (insbesondere Flash und Java-Applets), heutzutage bilden derart eingebettete Inhalte eher die Ausnahme (z.B. eingebettete PDF-Dokumente).
