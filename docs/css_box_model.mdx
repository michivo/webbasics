Wenn ein Browser ein HTML-Dokument rendert, muss er klarerweise wissen, *wie groß* jedes einzelne Element sein muss.
Im Browser werden alle Elemente mit Hilfe des **Box Models** modelliert. Alle HTML-Elemente können sozusagen als
"Boxen" betrachtet werden, die vom Browser gezeichnet werden. Jede dieser Boxen hat:
- eine Höhe und Breite (`height` und `width`), die die Abmessungen des Inhalts beschreiben
- den Innenabstand (`padding`) zwischen dem Inhalt und dem Rahmen
- einem optionalen Rahmen (`border`), der die Box umgibt
- einem Außenabstand (`margin`), der den Abstand außerhalb des Rahmens festlegt

Für all diese Teile der Box gibt es entsprechende CSS-Eigenschaften.
